# Power Function

#### Problem Description

Implement **pow(A, B) % C**

In other words, given that **A, B and C**. Find (**A^B % C**)

**Note:**

* The remainders on division cannot be negative. In other words, make sure the answer is positive

#### Problem Constraints

```
-10^9 <= A <= 10^9
0 <= B <= 10^9
1 <= C <= 10^9
```

#### Input Format

* Given three integers - **A, B & C**

#### Output Format

Return an integer

#### **Example Input & Output**

| Sr. No. | Input               | Output |
| ------- | ------------------- | ------ |
| 1       | A = 2, B = 3, C = 3 | 2      |
| 2       | A = 3, B = 3, C = 1 | 0      |

#### Example Explanation

Explanation 1:

```
2^3 = 8 => 8 % 3 = 2
```

Explanation 2:

```
3^3 = 27 => 27 % 1 = 0
```
